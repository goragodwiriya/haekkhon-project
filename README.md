# Haekkhon PHP Framework
จิ๋วแต่แจ๋ว เข้าใจตรง
เบา บาง และไม่มีอะไรมากมาย สร้างขึ้นเพื่อให้คุณเข้าใจหลักการทำงานของ PHP Framework แบบพื้นฐานที่สุด

เบื่อกับ Framework ใหญ่ๆ ที่ซับซ้อน? ตัวนี้คือคำตอบสำหรับการเรียนรู้และทำโปรเจคเล็กๆ ที่ไม่ต้องการความยุ่งยาก

## ✨ จุดเด่น
- 🚀 เร็วและเบา - โค้ดน้อย ทำงานไว
- 🧠 เข้าใจง่าย - ไม่มี Magic ที่ซ่อนอยู่ อ่านโค้ดได้ทันที
- 🛠️ ปรับแต่งได้เต็มที่ - อยากเพิ่มฟีเจอร์อะไร? เขียนเข้าไปตรงๆ เลย
- 🚀 เริ่มต้นใช้งาน

ดาวน์โหลด โปรเจคไปวางใน Web Server (เช่น htdocs)

ตั้งค่า Document Root ให้ชี้ไปที่โฟลเดอร์ public/

เปิดเบราว์เซอร์ ที่ URL ของคุณ

## 📖 วิธีใช้งาน

เปิดไฟล์ public/index.php แล้วกำหนดเส้นทางด้วยฟังก์ชัน route()

```php
<?php
require_once '../haekkhong.php';

// หน้าแรก
route('/', function() {
    echo "<h1>สวัสดีชาวโลก!</h1>";
});

// หน้าอื่นๆ
route('/about', function() {
    echo "<h1>เกี่ยวกับเรา</h1>";
});

// เริ่มต้นการทำงาน
dispatch();
```

## 📁 โครงสร้าง

```
haekkhong-framework/
├── public/
│   ├── index.php      # จุดเริ่มต้น กำหนดเส้นทาง
│   └── .htaccess      # สำหรับ URL Rewriting
└── haekkhong.php      # หัวใจของ Framework
```
ขอให้สนุก! 🚀
